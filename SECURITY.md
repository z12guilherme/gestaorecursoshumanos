# Política de Segurança

## Versões Suportadas

Atualmente, oferecemos suporte de segurança para as seguintes versões do projeto:

| Versão | Suportada          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reportando uma Vulnerabilidade

Levamos a segurança deste projeto a sério. Se você encontrar alguma vulnerabilidade ou problema de segurança, por favor, siga os passos abaixo:

1.  **Não abra uma issue pública** no GitHub para relatar falhas de segurança sensíveis.
2.  Envie um e-mail para `security@gestaorh.com` (email fictício para exemplo) com detalhes da vulnerabilidade.
3.  Inclua passos para reproduzir o problema, se possível.

Nossa equipe analisará o relatório e responderá o mais breve possível. Agradecemos sua colaboração para manter o sistema seguro.
